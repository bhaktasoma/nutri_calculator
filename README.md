# Nutrition Macro Calculator

A simple static nutrition macro calculator built for daily diet planning.

## What it does

- Estimates Basal Metabolic Rate (BMR) using the Mifflin-St Jeor formula.
- Adjusts BMR by activity level to compute Total Daily Energy Expenditure (TDEE).
- Provides a macro breakdown using a 25/30/45 split for protein, fat, and carbs.

## How to use

1. Open `index.html` in your browser.
2. Enter your age, gender, weight, height, and activity level.
3. Press `Calculate` to view calorie needs and macro targets.

## Notes

- This tool is an estimate and educational aid, not a substitute for medical or nutritional advice.
- For personalized plans, consult a registered dietitian or nutrition expert.
